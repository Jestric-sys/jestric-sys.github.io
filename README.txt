Проект по игре Battleship. Здесь описана хронология процесса разработки.

v0.1 - Начало проекта:
        1. Добавление UI:
            a. Экран поля
            b. Сообщения процесса игры
            c. Управление ввода данных
        2. Добавление объекта "Представление (view)":
            a. Метод добавления текстовой строки (displayMessage)
            b. Метод добавления попаданий на поле (displayHit)
            c. Метод добавления промахов на поле (displayMiss)

v0.2 - Изменения:
        1. Изменение UI:
            a. Добавлена кнопка версии игры
            b. Изменена форма ввода данных
        2. Добавление объекта "Метод (method)":
            a. Размер игрового поля
            b. Количество кораблей в игре
            c. Длина корабля в клетках
            d. Количество кораблей потопленных игроком
            e. Координаты и попадания кораблей
        3. Система:
            a. Добавлен препроцессор less для удобства пользованием css

v0.3 - Изменения:
        1. Изменение объекта "Представление (view)":
            a. Добавление метода выстрела игрока "fire"
            b. Добавление метода проверки потоплен корабль или нет "isSunk"

v0.4 - Изменения:
        1. Добавление объекта "Контроллер (controller)":
            a. Количество выстрелов
            b. Метод  получающий и преобразующей координат
        2. Добавлена новая функция "parseGuess" для правильности введенных игроков координат

v0.5 - Изменения:
        1. Добавление функции "init" для создание события кнопки "Огонь!"
        2. Добавление функции "handleFireButton" для вызова после нажатия на кнопку "Огонь!"
        3. Добавление функции обработки события при нажатии кнопки "Enter"
        4. Изменение UI:
            a. Изменен шрифт оповещение состояния игры "messageArea"

v0.6 - Изменения:
        1. Изменение UI:
            a. Добавлена ico в title
        2. Изменение объекта "Модель (model)":
            a. Добавление метода "generateShipLocations" - Создает массив "ships" с количеством кораблей, определяемым свойством "numShips"
            b. Добавление метода "generateShip" - Создает корабль в произвольном месте, не исключая перекрытия
            c. Добавление метода "collision" - Получает данные о местоположении корабля и проверяет, что он не перекрывается с кораблями, уже находящимися на поле
